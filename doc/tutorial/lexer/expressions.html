1.2.4 Adding Expressions

These are regular expressions used by the lexer rules.

The expression file syntax is described in this document.

To add expressions to the minilang language:

  • Right-click your project in the Solution Explorer and select 'Add | New Item...'.
  • Select Visual C++ | Utility | Text File (.txt) item and name the file minilang.expr .
  • Add the following code to the expression file:
    expressions minilang.expr
    {
        digit = "[0-9]";
        integer = "{digit}+";
        identifier = "{unicode_id_start}{unicode_id_cont}*";
        ws = "[\n\r\t ]";
        separators = "{ws}+";
    }
