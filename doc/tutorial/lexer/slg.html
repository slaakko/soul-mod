1.2.6 Creating a Lexer Project File

A lexer project file consists of references to token, keyword, expression and lexer files.

The lexer project file syntax is described in this document.

To add a lexer project file to minilang:

  • Right-click your project in the Solution Explorer and select 'Add | New Item...'.
  • Select Visual C++ | Utility | Text File (.txt) item and name the file minilang.slg .
  • Add the following code to the lexer project file:
    project minilang.slg;
    tokens <minilang.token>;
    keywords <minilang.keyword>;
    expressions <minilang.expr>;
    lexer <minilang_lexer.lexer>;
