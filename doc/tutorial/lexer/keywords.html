1.2.3 Adding Keywords

The minilang keywords, or reserved words, consist of strings for the statement selection, types and Boolean literals that are mapped to the corresponding tokens.

The keyword file syntax is described in this document.

To add keywords to the minilang language:

  • Right-click your project in the Solution Explorer and select 'Add | New Item...'.
  • Select Visual C++ | Utility | Text File (.txt) item and name the file minilang.keyword .
  • Add the following code to the keyword file:
    keywords minilang.keyword
    {
        ("if", IF), ("else", ELSE), ("while", WHILE), ("return", RETURN), ("int", INT), ("bool", BOOL), ("void", VOID), ("true", TRUE), ("false", FALSE)
    }
