Installation

Prerequisites:

  • Microsoft Visual Studio 2022 Community Edition with Desktop Development with C++ tool chain installed.

Do the following steps:

  • Extract the soul-5.0.0.zip package to a directory of your choice. In these instructions it is expected that the package is extracted to the C:\ root directory so that the instalation directory is C:\soul.5.0.0.
  • Create SOUL_ROOT environment variable and set it to contain path to the soul-5.0.0 installation directory:


    The SOUL_ROOT environment variable is used by Soul programs to find the Otava standard library std and the Unicode database soul_ucd.bin.
  • Create bin and lib directories under the soul-5.0.0 installation directory:
      Microsoft Windows [Version 10.0.26200.8246]
      (c) Microsoft Corporation. All rights reserved.

      C:\soul-5.0.0>md bin

      C:\soul-5.0.0>md lib

      C:\soul-5.0.0>
  • Add the bin directory to PATH environment variable:

  • Open the x64 Native Toools Command Prompt:

      **********************************************************************<
      ** Visual Studio 2022 Developer Command Prompt v17.14.30
      ** Copyright (c) 2025 Microsoft Corporation
      **********************************************************************
      [vcvarsall.bat] Environment initialized for: 'x64'

      C:\Program Files\Microsoft Visual Studio\2022\Community>
  • Change to the C:\soul-5.0.0\ext\zlib-1.2.11\contrib\masmx64 directory:
      C:\Program Files\Microsoft Visual Studio\2022\Community>cd \soul-5.0.0\ext\zlib-1.2.11\contrib\masmx64

      C:\soul-5.0.0\ext\zlib-1.2.11\contrib\masmx64>
  • Run bld_ml64.bat:
      C:\soul-5.0.0\ext\zlib-1.2.11\contrib\masmx64>bld_ml64.bat

      C:\soul-5.0.0\ext\zlib-1.2.11\contrib\masmx64>ml64.exe /Flinffasx64 /c /Zi inffasx64.asm
      Microsoft (R) Macro Assembler (x64) Version 14.44.35225.0
      Copyright (C) Microsoft Corporation. All rights reserved.

      Assembling: inffasx64.asm

      C:\soul-5.0.0\ext\zlib-1.2.11\contrib\masmx64>ml64.exe /Flgvmat64 /c /Zi gvmat64.asm
      Microsoft (R) Macro Assembler (x64) Version 14.44.35225.0
      Copyright (C) Microsoft Corporation. All rights reserved.

      Assembling: gvmat64.asm

      C:\soul-5.0.0\ext\zlib-1.2.11\contrib\masmx64>
  • Exit the x64 Native Toools Command Prompt:
      C:\soul-5.0.0\ext\zlib-1.2.11\contrib\masmx64>exit
  • Open C:\soul-5.0.0\soul.sln in Visual Studio.
  • Select the Debug|x64 configuration and build the solution.
  • Select the Release|x64 configuration and build the solution.
  • Search for x64 Native Toools Command Prompt, right-click its icon and select Open file location:
  • Right-click the x64 Native Toools Command Prompt icon and select Properties..:

  • Select the Target path and copy it to clipboard:

  • Open C:\soul-5.0.0\config\otava.config file to a text editor such as notepad or notepad++.
  • Paste the target path from clipboard as the value of the path attribute of the vcvars64 element:


    The vcvars64 path is used by the Otava compiler oc.exe to start a Visual Studio Developer Command Prompt and to find msbuild.exe.
  • Save the file and close the text editor.
